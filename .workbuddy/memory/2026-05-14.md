## 上传/发帖流程修复（上午 11:00-11:35）

### 问题链
1. **上传 404** → 前端手动设置 `Content-Type: multipart/form-data` 导致 boundary 丢失
2. **PUT update 400** → 后端强制要求 title+content，前端只传了 media 字段
3. **media_url 为 `undefined/posts/...`** → Cloudflare Worker 的 `R2_PUBLIC_URL` 环境变量未生效（Dashboard 可能覆盖了 wrangler.toml）
4. **所有帖子 media_url = null** → 因为 PUT 失败，数据库没写入媒体信息

### 修复
- **前端 `request/index.js`**：删除手动 Content-Type，让 Axios 自动处理
- **前端 `CreatePost.vue`**：update 时携带 title + content；加强错误日志和 alert
- **后端 PUT `/api/posts/:id`**：改为部分更新（只传 media 字段也行，不再强校验 title/content）
